Pede isto ao Replit AI Agent para evoluir o mapa para um verdadeiro editor fotovoltaico profissional:

---

Quero melhorar o módulo “Mapa Satélite” para suportar múltiplos perímetros/telhados e desenho completo do traçado das strings FV.

# 1. Suporte para múltiplos perímetros

Atualmente apenas existe um polígono/telhado.

Quero permitir:

* vários telhados;
* várias áreas;
* múltiplos polígonos independentes;
* múltiplas zonas de instalação.

Cada perímetro deve ter:

* ID único;
* nome;
* tipo:

  * coplanar
  * triângulos
  * solo;
* orientação;
* inclinação;
* nº painéis;
* strings associadas;
* potência associada.

---

# 2. Ferramenta de desenho de perímetros

Adicionar toolbar:

## Ferramentas:

* Desenhar polígono
* Editar perímetro
* Mover perímetro
* Duplicar perímetro
* Apagar perímetro
* Dividir área
* Unir áreas

---

# 3. Gestão independente por área

Cada área deve permitir:

* orientação própria;
* espaçamento próprio;
* tipo de estrutura próprio;
* modelo de painel próprio;
* strings independentes.

# 5. Desenho do traçado das strings

Quero um verdadeiro editor elétrico de strings.

Adicionar ferramenta:

```txt id="l6idk8"
Modo Strings
```

O utilizador deve poder:

* clicar nos painéis;
* criar sequência elétrica;
* desenhar percurso da string;
* definir início/fim;
* ligar strings a inversores.

---

# 6. Visualização das strings

Cada string deve:

* ter cor própria;
* mostrar número da string;
* mostrar:

  * tensão;
  * corrente;
  * potência;
  * nº módulos.

Exemplo:

```txt id="33l69j"
String 1
12 módulos
Voc = xxx
Vmp = xxx
```
